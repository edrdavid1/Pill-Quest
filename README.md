# Pill Quest PWA 🎮💊

Трекер приёма таблеток в стиле пиксель-арт с поддержкой PWA.

## 🚀 Возможности

### Основные
- ✅ Отметка о приёме таблетки
- 📅 Календарь с историей
- 📊 Статистика (стрик, всего дней, процент за месяц)
- 🔔 Напоминания в 8:00
- 🌓 Тёмная тема
- 💾 Экспорт данных

### PWA-фичи
- 📱 Установка на домашний экран
- 📴 Офлайн-режим
- 🔔 Push-уведомления
- ⚡ Быстрая загрузка

## 📦 Установка

### 1. Сгенерируйте иконки

Откройте `icons/icon-generator.html` в браузере и нажмите "Сгенерировать все иконки".

Или используйте скрипт (требуется librsvg или ImageMagick):
```bash
cd icons
./generate.sh
```

### 2. Запустите локальный сервер

PWA требует HTTPS или localhost:

```bash
# Python
python3 -m http.server 8000

# Node.js (если есть http-server)
npx http-server -p 8000

# PHP
php -S localhost:8000
```

### 3. Откройте в браузере

Перейдите на `http://localhost:8000`

### 4. Установите как PWA

**Chrome/Edge:**
- Нажмите на иконку ⋮ (меню)
- Выберите "Установить Pill Quest" или "Создать ярлык..."

**Safari (iOS):**
- Нажмите "Поделиться"
- Выберите "На экран «Домой»"

**Firefox:**
- Нажмите на иконку 🔧 в адресной строке
- Выберите "Установить"

## 📁 Структура проекта

```
pillows/
├── index.html           # Главная страница
├── styles.css           # Стили (светлая/тёмная тема)
├── app.js               # Основная логика приложения
├── sw.js                # Service Worker (офлайн, уведомления)
├── manifest.json        # PWA манифест
├── README.md            # Этот файл
└── icons/
    ├── icon.svg         # SVG иконка (векторная)
    ├── icon-generator.html  # Генератор PNG иконок
    ├── generate.sh      # Скрипт генерации (bash)
    └── icon-*.png       # PNG иконки (после генерации)
```

## 🎮 Использование

1. **Принять таблетку:** Нажмите большую кнопку "ВЫПИТЬ!"
2. **Просмотр истории:** Используйте календарь (◀ ▶ для навигации)
3. **Статистика:** Смотрите текущий стрик и процент выполнения
4. **Тёмная тема:** Нажмите 🌙 в правом верхнем углу
5. **Экспорт:** Нажмите "Экспорт данных" для сохранения истории

## ⚙️ Настройки

### Время уведомлений
Измените в `app.js` (строка ~13):
```javascript
settings: {
    notificationTime: '08:00',  // Формат ЧЧ:ММ
    theme: 'light'              // 'light' или 'dark'
}
```

## 🔧 Технические детали

### Хранение данных
- Данные хранятся в `localStorage`
- Ключи: `pill_quest_data`, `pill_quest_settings`
- Формат: `{ "YYYY-MM-DD": true }`

### Service Worker
- Кэширует все статические файлы
- Работает офлайн
- Обрабатывает уведомления

### Уведомления
- Ежедневные в заданное время
- Не показываются, если таблетка уже принята
- Требуют разрешения пользователя

## 🐛 Устранение проблем

### PWA не устанавливается
- Убедитесь, что используете HTTPS или localhost
- Проверьте, что `manifest.json` доступен
- Проверьте консоль на ошибки

### Уведомления не работают
- Дайте разрешение на уведомления
- Проверьте настройки браузера
- Service Worker должен быть активен

### Данные пропали
- Проверьте `localStorage` в DevTools
- Очистка кэша браузера удаляет данные
- Регулярно делайте экспорт
